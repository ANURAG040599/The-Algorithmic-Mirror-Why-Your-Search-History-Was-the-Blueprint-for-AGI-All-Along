## 📖 Overview

For years, recommendation engines tracked our every query, interaction, and logical leap — not just to sell products, but to map the structure of human cognition itself.

**The Algorithmic Mirror** is a research framework built on a single, testable hypothesis:

> *AGI emerges from the **Comparative Delta** — the measurable gap between human causal intent and algorithmic correlational prediction. Every failed recommendation, every user correction, every lateral search leap is a training signal no supervised dataset could manufacture.*

This repository contains the architecture, derivation engine, mathematical formulations, and benchmarks underlying the 2026 World Model system described in the [original essay](https://anuragdongare1.substack.com/p/the-algorithmic-mirror-why-your-search).

## 🧠 Core Thesis

Traditional LLMs learn **what** people look for.  
The Algorithmic Mirror framework learns **how** people think.

The key insight: recommendation failure is not noise — it is signal. Every time a system failed to anticipate a user’s next query, and the user corrected it, that correction demonstrated the gap between **correlational machine logic** and **causal human reasoning**.

Accumulated across billions of users and trillions of queries, this gap is the raw material for genuine machine cognition.

## 🏗️ System Architecture

The 2026 framework operates as a **Three-Phase Derivation Engine**:

### Phase 1 — Human Cognitive Pursuit

Captures years of search history vectors encoded not as topic clusters, but as **trajectories of curiosity** — how a mind moves from idea to idea, what logical bridges it builds, where it backtracks.

### Phase 2 — Audit & Derivation Engine

Analyzes **Predictive Leaps**: moments where the algorithm correctly anticipated the user’s next move before they made it. These convergence points reveal where correlational intelligence brushes against causal reasoning.

Synthesizes a **physics-grounded, multidimensional model of reality** capable of simulating outcomes internally before acting on them. Not a next-token predictor. A causal engine.

## 📐 Mathematics of Convergence

Intelligence is defined by the **contraction factor** of the latent space update operator.

Let `T` be the update operator that minimizes KL divergence between target human utility distribution `P_human` and the model’s current distribution `P_model`:

```
T: P_model → P_model'   such that   KL(P_human || P_model') < KL(P_human || P_model)
```

For the system to converge on a causal world representation:

```
‖T(P₁) - T(P₂)‖ ≤ c · ‖P₁ - P₂‖,   where c ∈ [0, 1)
```

**The smaller `c`, the faster the system’s internal representation converges to mirror the user’s causal model of the world.**

This is operationally what distinguishes GPT-class correlational predictors (high `c`, slow convergence) from 2026-generation World Models (low `c`, rapid causal alignment).

## 🚀 Getting Started

### Prerequisites

```bash
python >= 3.11
torch >= 2.4
lean4                    # For formal proof verification
numpy, scipy, matplotlib
```

### Installation

```bash
git clone https://github.com/anuragdongare/algorithmic-mirror.git
cd algorithmic-mirror
pip install -r requirements.txt
```

### Quick Start: Compute the Comparative Delta

```python
from architecture.derivation_engine import DerivationEngine
from math.delta_scoring import ComparativeDelta

# Initialize the engine with search history vectors
engine = DerivationEngine(protected_ledger="data/physics_laws.json")

# Load a cognitive trajectory
trajectory = engine.load_trajectory("data/sample_trajectories/physics_search.json")

# Compute the Comparative Delta
delta = ComparativeDelta(trajectory)
print(f"Contraction factor c: {delta.contraction_factor:.4f}")
print(f"Causal alignment score: {delta.alignment_score:.2%}")

# Run the World Model simulation
result = engine.simulate(trajectory, steps=100)
print(f"Simulation converged: {result.converged}")
print(f"Recovery accuracy (simulated corruption): {result.recovery_accuracy:.2%}")
```

### Run Benchmarks

```bash
# Self-healing accuracy benchmark
python benchmarks/self_healing_eval.py --corruption 0.72

# Causal alignment benchmark
python benchmarks/causal_alignment_score.py --dataset data/sample_trajectories/
```
